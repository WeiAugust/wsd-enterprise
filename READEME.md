# WSD All — 企业级 AI Coding 一站式交付框架

## 项目概述

基于 Claude Code 生态，集成六大开源项目精华，构建企业级 AI 编码交付解决方案。

**核心目标**：从需求提案到代码交付，全程由 AI 驱动，通过统一资产管理平台确保质量与一致性。

## 子项目

| 子项目 | 状态 | 描述 |
|--------|------|------|
| [wsd/](wsd/) | 开发中 | Claude Code 插件：需求生命周期管理 |
| [wsd_manager/](wsd_manager/) | 开发中 | 企业级 Claude Code 资产管理平台（部门/团队/仓库/个人四层） |
| [reference_projects/](reference_projects/) | 参考资料 | 六大开源项目源码 |

## 参考开源项目

| 项目 | 核心价值 |
|------|----------|
| **Everything Claude Code (ECC)** | 资产标准格式、插件市场体系、企业级控制 |
| **Get Shit Done (GSD)** | 阶段驱动执行、上下文守卫、工作区管理 |
| **Superpowers** | 技能自动触发、子代理驱动开发（SDD） |
| **Trellis** | 规格自动注入、多平台适配、项目记忆 |
| **OpenSpec** | 需求变更生命周期管理 |
| **Oh My ClaudeCode (OMC)** | 多代理团队编排、模型路由策略 |
